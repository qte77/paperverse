## Agent Guidelines

See `CONTRIBUTING.md` for core principles.

### Quality Thresholds

| Dimension | Minimum | Description |
|-----------|---------|-------------|
| Context | 8/10 | Accurate understanding of requirements |
| Clarity | 7/10 | Clear, unambiguous output |
| Alignment | 8/10 | Matches stated task scope |

### Mandatory Prohibitions

- [ ] Do not assume missing context — ask or research
- [ ] Do not hallucinate libraries — only use verified dependencies
- [ ] Do not skip validation — run `make validate` before completion
- [ ] Do not over-engineer — match complexity to task scope
